## üìè Image Specifications

- **Format**: PNG with transparent background
- **Main Cookie Size**: 128x128 pixels (recommended)
- **Building/Upgrade Icons**: 64x64 pixels (recommended)
- **Quality**: High resolution, clean edges

## üîß How It Works

1. Drop your images into the appropriate folder (`fortnite/` or `genshin/`)
2. Name them exactly as shown in the tables above
3. When you select that skin in Settings, the game will use your images
4. If an image is missing, it will fall back to the default emoji

## üìù Priority Items

**Most Important (do these first):**
1. **cookie.png** - This is what players click constantly
   - Fortnite: V-Bucks icon
   - Genshin: Mora coin
2. **grandma.png** - Most common building
   - Fortnite: Peely or any character
   - Genshin: Paimon

**Less Important:**
- The other buildings can use the default emojis if you don't have images

## üí° Where to Find These Images

- **Fortnite**: Search "Fortnite [item name] icon PNG" (e.g., "Fortnite V-Bucks icon PNG")
- **Genshin**: Search "Genshin Impact [item name] icon PNG" (e.g., "Genshin Impact Mora PNG")
- Look for wiki pages (Fortnite Wiki, Genshin Impact Wiki) which often have clean icon images
